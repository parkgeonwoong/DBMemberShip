## ✅ Purpose
1. 데이터베이스 시스템 구성 및 DBMS를 이해할 수 있다.
2. 관계-데이터 모델 개념 및 관계 대수를 이해할 수 있다.
3. SQL 기초 및 고급 이해하고 SQL을 이용한 DB 실습을 할수 있다.
4. 웹기반 데이터베이스 프로그래밍 방법을 이해하고 실습할 수 있다.

<br />

## ✅ Environment
- Server : Apache Tomcat 9.0 Tomcat9
- DateBase : Oracle SQL Developer
- 사용 언어 : Java, JSTL, JavaScript
- 실행 환경: VSCode & Eclipse IDE for Enterprise Java and Web Developers

### 🔸 Project Requirements
```markdown
 1. 회원 인증 및 회원관리(가입, 수정, 삭제 기능)의 전체적인 소프트웨어 구조
 2. 회원 정보 관리를 위한 DB 스키마 설계
 3. 회원 가입 HTML FORM 문서 만들기 및 구현 구조
     - 회원 가입 폼 작성 페이지 설계
     - 중복 체크 확인 페이지 설계
 4. 회원 관리(가입, 수정, 삭제) DB 저장 페이지 구현
 5. 회원 관리(가입, 수정, 삭제) 도중 필요한 자바스크립트 작성
 6. 회원 인증 기능 구현
     - 회원 인증 폼 작성 페이지
     - 로그인 처리 페이지
     - 로그아웃 처리 페이지
 7. 회원 인증 및 관리에 필요한 자바빈즈 설계 및 구현
 8. 회원 인증 및 가입 페이지 실행
 ```
